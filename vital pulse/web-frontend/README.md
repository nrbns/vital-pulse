# Pulse Web Frontend

Web preview of the Pulse mobile app UI/UX. This allows you to see and test the UI in a browser without needing Android Studio or Xcode.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The web app will be available at **http://localhost:3001**

## Features

- Preview of all mobile app screens
- Regional adaptation (colors, languages)
- Interactive UI components
- Responsive design (mobile-first)
- Connected to backend API (http://localhost:3000)

## Notes

This is a **preview/demo** version of the mobile app UI. The full React Native mobile app is in `/mobile-app`.
